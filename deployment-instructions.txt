Deployment Instructions for mkrl

Follow these steps to deploy the application to Vercel

1. Import Project
- go to your vercel dashboard and click add new project
- import the repository containing this code
- enter the project name as mkrl

2. Configure Project Settings
- in the build and development settings section set the following
- build command: npm run vercel-build
- output directory: frontend/dist
- install command: npm install

3. Environment Variables
- add the following environment variables in the settings
- DATABASE_URL: your postgresql database connection string
- REDIS_URL: your redis connection string

4. Deploy
- click deploy to start the process
- once complete check that both the frontend and api are working

Note
- ensure your database and redis instances are accessible from vercel
